# Project Title

[Vite](https://vitejs.dev) and [PostCSS](https://github.com/postcss/postcss).

## Run Locally

1. Download or clone the project

   ```sh
   git clone https://github.com/
   ```

2. Go to the project directory

   ```sh
   cd folder-name
   ```

3. Install dependencies

   ```sh
   yarn install
   ```

4. Start the local server

   ```sh
   yarn dev
   ```
